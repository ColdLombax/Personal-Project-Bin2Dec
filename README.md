# Bin2Dec
A simple website to convert binary (BASE 2) into decimal (BASE 10) and visa versa.
#### The Good
- Happy with Color Scheme.
- Error Handling

#### The Bad
- Jumpy on Mobile.
- Binary Remains with Error.
